# H VS M - Hack vs Myth App

## App Description
H VS M is an Android quiz app that tests users on common myths and useful hacks. The user is shown a statement and must choose whether it is a Hack or a Myth.

## Features
- Splash screen with logo
- Hack and Myth answer buttons
- Feedback after selecting an answer
- Next button to move through questions
- Score tracking
- Custom background and UI
- Logging using Log.d() to show app actions in Logcat

## How the App Works
The app stores myth and hack questions in an array. Each question has a correct answer. When the user selects Hack or Myth, the app checks the answer, displays feedback, updates the score, and logs the action in Logcat.
